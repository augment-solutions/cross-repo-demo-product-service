# Event-Driven Common Library (TypeScript)

Shared TypeScript library for event-driven microservices.

## Features

- Redis Streams client
- Event envelope with standard format
- Datadog tracing integration
- Structured logging with Winston

## Installation

```bash
npm install
```

## Usage

See individual service implementations for usage examples.

## Building

```bash
npm run build
```

## Testing

```bash
npm test
```
